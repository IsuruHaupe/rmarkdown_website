Here is a quick example of what can be done with the package FEstim:

Step 1 : Creation of the bed matrix

require("FEstim")
bed_matrix <- read.bed.matrix("Examples/geb")

Step 2 : creation of the segments

Segments <- segments(bed_matrix)
segment.summary(Segments)

Step 3 : creation of the submaps

Submaps <- submaps(bed_matrix, 100, Segments)

Step 4 : plotting

HBD.plot.chr(Submaps, chr=1)
HBD.plot.id(Submaps, individual.id="A011", family.id="0")
